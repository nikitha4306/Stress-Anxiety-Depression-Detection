# Stress Anxiety Depression Detection (DASS-21)

## Project Overview
This project is a Machine Learning based web application that predicts Stress, Anxiety, and Depression levels using the DASS-21 questionnaire.

The system uses trained ML models to analyze user responses and classify mental health conditions.

## Features
- DASS-21 questionnaire interface
- Predicts Stress level
- Predicts Anxiety level
- Predicts Depression level
- Web-based interface using Flask
- Machine Learning models integrated

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Scikit-learn
- Joblib
- Pandas
- NumPy

## How to Clone and Run the Project

### 1. Clone the Repository
```bash
git clone https://github.com/nikitha4306/Stress-Anxiety-Depression-Detection.git
```

### 2. Navigate to the Project Folder
```bash
cd Stress-Anxiety-Depression-Detection
```

### 3. Install Required Libraries
```bash
pip install -r requirements.txt
```

### 4. Run the Flask Application
```bash
python app.py
```

### 5. Open in Browser

Open your browser and go to:

http://127.0.0.1:5000/
